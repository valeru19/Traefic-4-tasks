Подключение сертификата для работы по https
• Узнайте fqdn Вашего компьютера
• Сгенерируйте пару ключ: сертификат для доменного имени вашего компьютера
(https://regery.com/en/security/ssl-tools/self-signed-certificate-generator)
• Скопируйте ключ и сертификат в соответствующие файлы и подмонтируйте их к контейнеру в
соответствии со схемой файловой структуры
• Настройте traefik на использование сертификатов
